Settings

Fine-tune every detail

The Settings tab puts you in full control — no config files to edit by hand.

🎛️ General Options

  • → Folder Prefix — prepend a custom label to all destination folders (default: Sifter-)
  • → No Prefix — check the box to sort into plain folder names with no prefix
  • → Move Folders — move subdirectories into a dedicated Folders subfolder
  • → Add Category — create a brand-new category with a custom name and extension list
  • → Reset Defaults — restore all categories and settings to factory defaults

⚠️ Advanced: Recursive Mode

  • ! When enabled, WinSifter descends into every subfolder and sorts files found there too.
  • ! This is a destructive, hard-to-reverse operation on large directory trees.
  • ! Always run a Preview first when using recursive mode.
  • ! Hidden behind an expandable panel — must be deliberately unlocked.
